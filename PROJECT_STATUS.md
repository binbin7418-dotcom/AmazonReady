## 💡 关键决策

1. **使用 Zustand 而不是 Context API**
   - 更简单的状态管理
   - 更好的性能
   - 更少的样板代码

2. **使用 react-dropzone**
   - 开箱即用的拖拽上传
   - 文件验证
   - 良好的用户体验

3. **使用 Sonner 而不是自定义 Toast**
   - 美观的 UI
   - 简单的 API
   - 轻量级

4. **数据库触发器自动创建用户档案**
   - 简化注册流程
   - 保证数据一致性
   - 自动分配免费积分
